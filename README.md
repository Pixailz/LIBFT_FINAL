## USAGE

```bash
make clean
make fclean
make re
make help
make <part>
make <part1|part2 ... partN>
```

> do `make help` to have a short summary of all part

- available part are:
	- integer
	- string
	- memory
	- check
	- list
	- print
	- input
	- random
	- linux
	- error
